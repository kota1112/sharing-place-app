## サービスの概要
観光地やお店を共有するアプリ。行ったお店や行きたいお店を共有したり、自分だけのリストを作成できるアプリです。

## アプリを作ったキッカケ
オーストラリアに留学して3年、僕はメルボルンのたくさん主要な観光地に行きました。そしてネットに検索するだけでは情報が足りなくなりました。色んな人に会うたびに色んな体験、行った場所の話をし合いました。そこで僕はより多くの人に行った場所、お店の話を聞きたいと想いこのアプリを作ろうと思いました。また僕だけの願望だけでなく会った人、SNSでよくおすすめの観光地、レストラン、カフェについて聞いている人がいるのでより多くの情報がまとまっているサイト、アプリはオーストラリアを訪れる人にとって有益だと想い作りました。

## サービスへの想い
オーストラリアに訪れる人により良い体験をして欲しい、オーストラリアに限らず何気なく歩いてるとーのお店があった。美味しそうなお店があったこの景色は絶景だったなどいい思い出が沢山見つかると思います。その思い出、経験を他の人と共有してみんなが良い思い出を作って欲しいと思います。
