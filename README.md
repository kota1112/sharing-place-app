## サービスの概要
観光地やお店を共有するアプリ。行ったお店や行きたいお店を共有したり、自分だけのリストを作成できるアプリです。

## アプリを作ったキッカケ
オーストラリアに留学して3年、僕はメルボルンのさまざまな観光地やスポットを巡ってきました。
しばらくすると、ネットで調べるだけでは物足りなくなってきて、「実際に行った人の、生の声をもっと知りたい」と感じるようになりました。

メルボルンで出会う人たちと話すたびに、お互いの体験や、おすすめの場所・お店の話で盛り上がります。そこで、もっといろいろな人の「行ってよかった場所」の話を聞いてみたいと思い、このアプリを作ろうと決めました。

また、これは自分の願望だけではなく、実際に会った人や SNS 上でも、
「おすすめの観光地は？」「この辺で良いレストラン / カフェはある？」といった質問をよく目にします。
そんなときに、オーストラリアのスポット情報がまとまっていて、実体験ベースの声が集まっているサービスがあれば、旅行者や留学生にとって必ず役に立つと思い、このアプリを作りました。

## サービスへの想い
オーストラリアを訪れる人に、できるだけ良い体験をしてほしいという思いがあります。
オーストラリアに限らず、ふと歩いていて見つけたカフェ、たまたま入ったレストラン、何気なく見上げたときの絶景など、日常の中にもたくさんの「お気に入りの瞬間」があると思います。

そうした 「行ってみてよかった場所」「自分にとって特別になった景色」 を、誰か一人の中だけに閉じ込めておくのではなく、他の人とも共有できる場を作りたい。

このアプリを通して、
自分の思い出や経験をシェアする人も、それを参考に新しい場所に出かける人も、みんなが良い思い出を増やしていけるようにしたい──そんな思いでサービスを作っています。

## 今後の展望
本プロダクトは5つのフェーズに分けて、段階的に開発を進めています。
- フェーズ1：お気に入りボタン、お気に入り表示。タグ、フィルター機能（観光地、レストラン、カフェ等）
- フェーズ2：特集、情報の精度の高いサイトを追加。英語対応にする。
- フェーズ3：複数同じお店に対する投稿があった時の複数レビュー見れるようにする。無限スクロール機能
- フェーズ4：AIに尋ねる機能。
- フェーズ5: SNS化、フォロー機能。フォローした人が行った場所をフィルターして表示する機能。
